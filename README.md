# 🛒 Shopsy – E-commerce Web App

A modern **E-commerce web application** built using **React, Redux Toolkit, and Tailwind CSS**.
This project allows users to browse products, add items to cart, and manage their shopping experience.

---

## 🚀 Features

* 🛍️ Product listing with dynamic UI
* 🛒 Add to cart functionality
* ➕ Increase / decrease product quantity
* ❌ Remove items from cart
* ⚡ State management using Redux Toolkit
* 🎨 Responsive UI with Tailwind CSS
* 🔄 Real-time updates in cart

---

## 🛠️ Tech Stack

* ⚛️ React.js
* 🧠 Redux Toolkit
* 🎨 Tailwind CSS
* ⚡ Vite
